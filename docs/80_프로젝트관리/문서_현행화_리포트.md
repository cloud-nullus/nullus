## 미해결 이슈
- 문서 본문에 API 정책상 필요한 `v1` 표기는 일부 유지했다.
  - 예: `/api/v1/*` 엔드포인트 경로
  - 이유: 문서 버전 표기가 아닌 API 계약 경로이므로 제거 대상 아님
- 일부 대용량 설계 문서는 코드 블록 내 정규식 패턴이 링크 검사 도구에 오탐을 유발한다.
  - 운영 영향 없음, 링크 렌더링 기준 확인 필요
